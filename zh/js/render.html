    总结步骤：

处理 HTML 标记，构建 DOM 树

处理 CSS 标记，构建 CSSOM 树

将 DOM 树和 CSSOM 树融合成渲染树

根据生成的渲染树，计算它们在设备视口(viewport)内的确切位置和大小，这个计算的阶段就是回流 => 布局（Layout）或 重排（reflow）

根据渲染树以及回流得到的几何信息，得到节点的绝对像素 => 绘制（painting）
